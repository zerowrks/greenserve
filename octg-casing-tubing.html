Oil Country Tubular Goods (OCTG) – Casing and Tubing

Greenserve supplies OCTG casing and tubing and associated tubular products and other OCTG accessories, in accordance with API 5CT latest edition at the time of manufacture (PSL-1, PSL-2, PSL-3).

Type

  • Seamless and welded

Sizes

  • Outside Diameter: 1.66” to 30”.
  • All API Standard Weights
  • Length: Range 1, Range 2 and Range 3

Steel Grades

  • All API Grades: H40 to Q125 including chromium grades

Connections

  • Casing: Short round Thread Casing (STC), Long round Thread Casing (LTC), Buttress Thread Casing (BTC), Extreme-Line Casing (XL)
  • Tubing: Non-Upset Tubing (NUE), External Upset Tubing (EUE), Integral Joint Tubing (IJ)
  • Premium connection

Accessories

  • Pup Joint
  • Coupling
  • Blast Joint
  • Crossover, etc.
